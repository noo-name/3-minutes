Résumé du projet :

Vous arrivez devant un sol éclairé par une lumière, créant un cercle. À gauche, sur une table, vous pouvez mettre le casque de réalité virtuelle. Après l'avoir mis, vous vous retrouvez dans un menu qui vous mènera à un jeu où vous devrez, dans un gymnase, touchez le plus de cibles possible avec des ballons en 3 minutes. Quand ce temps sera écoulé, vous allez devoir déposer le casque et vous diriger vers le couloir derrière vous. Vous allez devoir entrer dans un casier et y restez 3 minutes. Des sons d'amusement, de claustrophobie et de peur, vous transporteront dans la peau d'un jeune intimidé. Le 3 minutes passé, sortez du casier pour lire un message de sensibilisation à l'intimidation écrit sur deux moniteurs.
